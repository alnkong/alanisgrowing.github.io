从Windows PC向M1 Mac迁移体会

2021-01-16 2 min read # Mac # Apple # Windows # technology # software # hardware # Gridea

这两天一直在弄自己的M1 Mac Mini，下面就说说自己的体会。

先说说问题

  1. Yamaha UR22 声卡驱动貌似不支持M1，只支持英特尔，截至2021年1月15日。
  2. Brother MFC-7880DN一体机接在路由器上，但是安装驱动后只显示打印功能，不显示扫描功能。
  3. 24寸显示器比起我之前的43寸带鱼屏加上28寸竖放显示器，真的小太多了。屏幕太小，经常需要切换啊，分屏啊之类的。
  4. 软件安装有点麻烦，需要解压，打开，拖拽，很多时候还有限制，报错，最后还要专门解锁开一堆权限，虽然要开很多权限的软件并不多。目前最麻烦的莫过于向日葵，那权限。。。

再说说优点

  1. iCloud挺香，极大幅度减少优盘使用次数，而且自己的日历、备忘录、照片和文件在各个设备上都有，极大程度提升安全感，不用惦记总是优盘同步，把优盘带到这里带到那里了。当然目前工作单位对需要网络的云存储支持还不好，所以工作依然需要用优盘。
  2. 苹果的显示技术真的很香，看着这个4k显示器，怎么看怎么舒服，细腻，悦目。

说说这几天的变化和惊喜

  1. 触控板刚开始不会用，现在用习惯了觉得还挺舒服的，滑滑的，手指在上面移来移去的。
  2. 博客写作工作Gridea我刚开始以为无法安装，后来又操作了好几遍，发现居然可以安装上，可以正常使用，也是意外的惊喜了。
  3. Photoshop一开始不确定能不能使用，结果居然正常安装上了，初步使用没问题。

总结

只要不玩游戏、不搞学术、不用行业软件，只做一般的办公、写作、学习、视频、音乐、图像处理，Mac挺好的。一开始提到的硬件问题希望随着时间的推移能够解决。这么多年同步优盘、想办法做时间管理、微信来来回回传图片、传文件、传通知，真的是累了。
